Here are our other accounts:

Action Network

Action Network is a service for driving campaigns and recruitment. It can do appeals, polls, fundraising, contact management, a bunch. We are using it for collecting onboarding information from new users.

Substack blog

It is easy to set up a substack account and the company has a good track record of resisting take-down requests. Substack has a built-in mailing list and a flexible content management system permitting multiple authors and editors.

Website

David Mintz set up a domain, freepalestine.org, and a simple landing page that points to our linktree page. Having a domain should increase search hits on our name, and we may want to build a full-featured website at some point.

Cloud Storage

We are currently using pCloud, which offer generous amounts of free storage. We also have a Google Account which we are using exclusively for Google forms when needed, though we could use the account for YouTube and other purposes.

Linktree

Linktree creates a "landing page" where links to an organization's social media accounts are created. When you update your social media you only have to update your linktree. Linktree also contains an onboarding link for new members.

Listserv

groups.io listserv software does not require users to have a Google account in order to use the services. Administration services are more flexible and the features are are better.

Messaging

40 people belong to the main "Gaza" Signal group and there are additional groups for Media, Naming, Next Steps, Marshals, and Speakers. Signal is not ideal as a main communication medium because subgroups are not used adequately and the main group becomes saturated with chitchat. Important messages are frequently lost or need to be repeated. The Next Steps subcommittee investigated using Discord as an alternative to Signal and tested it. A Tech committee should continue looking at it.

Password vault

Bitwarden is a good, cloud-based password manager/vault. All of our passwords are in this account. The same data is mirrored in a separate Avira password vault.

Social Media

We are currently using (Meta) Instagram, which shares an account with Meta's new Twitter/X competitor, Threads. We could use threads if we wanted.
