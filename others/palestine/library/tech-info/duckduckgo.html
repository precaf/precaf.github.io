DuckDuckGo email forwarding

We currently have a DuckDuckGo email alias that forwards email to our public address:

afreepalestine@duck.com freepalestine@disroot.org

Rather than use "afreepalestine@duck.com" you can generate any number of aliases using a web extension.

Here is the version for Chrome:
